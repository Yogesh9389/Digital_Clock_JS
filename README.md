# Digital Clock Project

This project implements a digital clock using HTML, CSS, and JavaScript.

## Features

- Displays the current date and time in real-time.
- Updates the clock every second to reflect the accurate time.

## Technologies Used

- HTML
- CSS
- JavaScript
